Machine Learning

Wat is Machine Learning

Machine Learning is een vorm van kunstmatige intelligentie (AI) die is gericht op het bouwen van systemen die van de verwerkte data kunnen leren of data gebruiken om beter te presteren. Je hebt twee soorten methodes die je kan verdelen in categorieën, namelijk aanleidinggevend en deductief. Met de aanleidinggevende methodes creëer je computerprogramma’s. Met de deductieve methodes heb je een functie net zoals de invoerdata. Machine Learning is gerelateerd aan statistiek. Het is ook gericht op algoritmische complexiteit. Het wordt ook gelinkt met datamining, dat is waarin op een geautomatiseerde manier patronen en relaties worden gezocht.
